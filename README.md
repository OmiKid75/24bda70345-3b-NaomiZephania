# Experiment 3B - To build a library management system with book search, add, and remove functionality.

## Name
Naomi Zephania

## UID
24BDA70345

## Objective
- Create responsive book listing
- Implement search functionality
- Add form for new book entries
- Enable book removal
- Manage state with React hooks

## Technologies Used
- Software: Node.js 18+, React 18+, VS Code
- Hardware: i5+ CPU, 8GB RAM

## Features
- Add new books with title and author
- Real-time search (case-insensitive filtering)
- Edit existing book details
- Remove books from the list
- Dynamic state management using useState
- Clean and modern UI using Tailwind CSS
- Reusable custom button component (LibraryButton)
- Fast development with Next.js App Router
- Type-safe development using TypeScript
